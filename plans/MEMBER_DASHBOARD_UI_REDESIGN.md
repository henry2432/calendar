# Kayarine Member Dashboard UI 重設計方案

**版本**：1.0  
**日期**：2026-02-01  
**狀態**：等待審核  
**實現方式**：PHP/WordPress + 現代化 HTML/CSS/JS（Elementor 兼容）  
**設計風格**：蘋果簡約留白 + 品牌活力橙

---

## 🎯 目標

基於 `fig-tem1/` 參考設計，為 Kayarine 會員中心重新設計 UI/UX，融合蘋果簡約留白美學與品牌活力橙色，保持 WordPress 生態相容，提升視覺現代感和互動體驗。

## 🎨 設計規範

### 核心美學原則

| 原則 | 實踐方式 |
|------|---------|
| **簡約** | 移除冗餘元素，每個區塊獨立、清晰 |
| **留白** | 60px+ 段落間距，寬裕的內邊距 |
| **聚焦** | 品牌活力橙作唯一強調色 |
| **對稱** | 網格對齊，卡片均勻分佈 |
| **光線感** | 淡陰影、無邊框或極細邊框 |

### 色彩系統

| 用途 | 顏色 | Hex | 說明 |
|------|------|-----|------|
| **主色/品牌色** | 活力橙 | `#FF7F00` | 按鈕、強調、重要互動 |
| **主色-暗** | 深橙 | `#E67E00` | 懸停、按下狀態 |
| **成功** | 綠色 | `#27ae60` | Completed 狀態 |
| **警告** | 橙色 | `#f39c12` | Pending 狀態 |
| **進行中** | 天藍 | `#3498db` | Processing 狀態 |
| **錯誤** | 紅色 | `#e74c3c` | Cancelled 狀態 |
| **背景** | 純白 | `#FFFFFF` | 頁面背景 |
| **卡片淡** | 超淡灰 | `#F9F9F9` | 卡片背景 |
| **邊界** | 極淡灰 | `#E8E8E8` | 邊框、分隔 |
| **文字深** | 深灰 | `#1F1F1F` | 標題 |
| **文字淡** | 中灰 | `#757575` | 副文本 |

### 排版規範

| 層級 | 字號 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| **H1** | 32px | 300 | 1.2 | 頁面標題（輕薄） |
| **H2** | 24px | 400 | 1.3 | 區塊標題 |
| **H3** | 18px | 500 | 1.4 | 卡片標題 |
| **Body** | 15px | 400 | 1.6 | 正文 |
| **Small** | 13px | 400 | 1.5 | 輔助文本 |
| **Label** | 12px | 500 | 1.4 | 標籤 |

**字型棧**：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;`

### 間距規範

| 尺度 | 值 | 用途 |
|------|-----|------|
| **xs** | 4px | 組件內部 |
| **sm** | 8px | 項目間距 |
| **md** | 12px | 標準間距 |
| **lg** | 16px | 區塊內邊距 |
| **xl** | 20px | 主要分隔 |
| **2xl** | 30px | 卡片邊距 |
| **3xl** | 40px | 段落間距 |
| **4xl** | 60px | 大區塊分隔 |

### 視覺特性

| 元素 | 規格 |
|------|------|
| **圓角** | 8px（卡片）/ 4px（按鈕）/ 50%（頭像） |
| **陰影** | 極淡（0 1px 3px rgba(0,0,0,0.05)） |
| **邊框** | 1px solid `#E8E8E8`（極淡） |
| **過渡** | 0.2s ease（平滑） |

## 📱 響應式設計

### 斷點規範

| 設備 | 寬度 | 佈局特性 |
|------|------|---------|
| **手機** | < 768px | 單欄，堆疊排列，邊距 16px |
| **平板** | 768px-1024px | 2 欄網格，邊距 24px |
| **桌面** | > 1024px | 3-4 欄網格，邊距 32px |

### 響應式調整

- **歡迎卡片**：手機 flex-column | 桌面 flex-row
- **預約列表**：手機 卡片堆疊 | 桌面 表格型 |
- **忠誠度面板**：手機 1 列 | 平板 1 列 | 桌面 2 列
- **推薦商品**：手機 1 列 | 平板 2 列 | 桌面 4 列

## 🧩 組件詳細規格

### 1. 歡迎卡片 (Welcome Card)

**高度**：240px (desktop) / 300px (mobile)  
**背景**：`#FFFFFF`（純白，無漸層）  
**邊框**：無或 1px solid `#E8E8E8`（極淡）  
**邊距**：32px（desktop）/ 16px（mobile）  
**陰影**：`0 1px 3px rgba(0,0,0,0.05)`（極淡）

**結構**（蘋果風格的簡潔）：
```
┌─ 頭像區 (左/上)
│  ├─ Avatar: 128px (desktop) / 80px (mobile)
│  └─ 編輯按鈕：50% border-radius, #FF7F00 背景
├─ 空白：16px
├─ 內容區 (右/下)
│  ├─ 標題：「歡迎回來」（淺灰色，300 font-weight）
│  ├─ 名稱：「[姓名]」（深灰，600 font-weight）
│  ├─ 成就：「今年已出海5次」（中灰 12px）
│  ├─ 空白：16px
│  ├─ 積分進度條
│  │  ├─ 標籤：「積分」（淡灰 12px）
│  │  ├─ 進度：活力橙色進度條（高 4px）
│  │  └─ 助文：「150 points to Gold」（淡灰 12px）
│  ├─ 空白：16px
│  └─ 按鈕組（水平排列）
│     ├─ 編輯資料（活力橙 outline）
│     ├─ 查看成就（outline）
│     └─ 會員專享（outline）
```

### 2. 我的預約 (Bookings List)

**標題邊距**：24px top, 8px bottom  
**卡片高度**：120px (desktop) / auto (mobile)

**預約卡片結構**（簡約設計）：
```
預約卡片
├─ 狀態標籤（top-right，活力橙或狀態色）
├─ 左側
│  ├─ 預約標題（18px, 600 weight）
│  ├─ 地點（13px, 中灰，with MapPin icon）
│  └─ 日期/時間（13px, 淡灰，with Calendar/Clock icon）
├─ 右側
│  ├─ 金額（18px, 深灰, 600 weight）
│  └─ 操作按鈕
│     ├─ 改期（活力橙 outline）
│     └─ 取消（紅色 outline）
```

### 3. 忠誠度面板 (Loyalty Panel)

**網格**：2 欄 (desktop) | 1 列 (mobile)  
**卡片尺寸**：equal  
**卡片高度**：140px

**卡片設計**（蘋果極簡）：
```
┌─────────────────────┐
│ 標籤（12px, 淡灰）   │
│                     │
│ 數值（32px, 深灰）   │
│                     │
│ 單位（12px, 淡灰）   │
└─────────────────────┘
```

### 4. 推薦商品 (Recommended Products)

**背景**：`#F9F9F9`（超淡灰）  
**網格**：1-4 列（響應式）  
**卡片**：簡潔、圖片為主

## 🔧 技術實現細節

### PHP 層

**檔案**：`kayarine-booking/includes/class-kayarine-member-dashboard.php`

**變更**：
1. 保留現有 AJAX 邏輯
2. 提取用戶數據（會員資料、積分、預約列表）
3. 新增數據適配函數
4. 保留短代碼 `[kayarine_member_dashboard]`

### HTML/CSS 層

**檔案**：`kayarine-booking/assets/css/style.css`（附加規則）

**特性**：
- CSS Grid + Flexbox 混合佈局
- 蘋果風格簡約配色
- 寬裕留白間距
- 平滑過渡動畫（hover 效果）

### JavaScript 層

**改進**：
1. 使用 Fetch API 替代 jQuery
2. Loading 狀態反饋
3. 無刷新更新 DOM
4. 現代化錯誤提示
